- 과제내용
- 아이디 또는 패스워드가 잘못 입력되었을 때 에러 메시지를 출력
- MessageBox를 사용하지 않고, 입력창 근처에서 오류를 표시하도록 구현
- Label 컨트롤과 Visible 속성을 활용하여 메시지 표시 및 숨김 기능 구현 
- 사용자 경험(UX)을 고려한 에러 메시지 위치 및 표현 방식 개선

- 구현내용과기능설명
- Label을 이용한 에러 메시지 구현
에러 메시지를 출력하기 위해 Label 컨트롤(lblErrorMsg)을 추가
기본 상태에서는 보이지 않도록 설정
- 로그인 실패 시 메시지 표시
lblErrorMsg.Visible = true;
아이디 또는 비밀번호가 틀렸을 경우 Label을 화면에 표시
입력창 근처에서 바로 오류 확인 가능
- 로그인 성공 시 메시지 숨김
lblErrorMsg.Visible = false;
로그인 성공 시 에러 메시지를 다시 숨김 처리
